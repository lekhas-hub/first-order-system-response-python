# First Order System Response (Python)

## 📌 Description
This project simulates the step response of a first-order control system.

## 🚀 Features
- Calculates system response over time
- Based on exponential function
- User-defined time constant

## 🧠 Concepts Used
- Control Systems
- First-order systems
- Step response
- Python math functions

## ▶️ How to Run
Run:
python main.py
